How to read this. Both tickers run identical grid bots ($1,000 × 4× leverage, 8 grids, 2% band by default) around the $2,200 anchor. They differ only in their starting inventory: Ticker A holds the equivalent of $5,000 long, Ticker B holds $5,000 short. The grid bots fire the same fills on both tickers (same price, same logic), but the effect on each ticker's PnL diverges because of the different starting positions. Unrealized PnL reflects each ticker's current mark-to-market against its current avg entry. Realized PnL accumulates from closed grid round-trips. The combined panel shows whether the two-position book is net winning or losing as a system.
